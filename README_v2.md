# Oxford-IIIT Pet Project (v2)

Wersja izolowana od konfliktowych plików.

## Pliki
- `pet_classification_project_v2.ipynb` — pełny notebook projektu.
- `frontend_app_v2.py` — frontend Streamlit.
- `requirements.txt` — zależności.

## Uruchomienie
1. Instalacja:
```bash
pip install -r requirements.txt
```

2. Otwórz i uruchom wszystkie komórki notebooka:
```bash
pet_classification_project_v2.ipynb
```

3. Uruchom frontend:
```bash
streamlit run frontend_app_v2.py
```

## Co dostajesz
- Analiza datasetu Oxford-IIIT Pet.
- 3 eksperymenty (bazowy -> ulepszenie -> ulepszenie poprzedniego).
- Metryki numeryczne i wykresy.
- Confusion matrix i przykładowe predykcje.
- Test modeli na własnym obrazie w frontendzie.
